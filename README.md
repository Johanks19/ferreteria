# 🛠️ Sistema de Tienda de Herramientas en Java

Sistema que permite la venta y alquiler de herramientas, gestionando clientes, inventario y transacciones.

## 🚀 Funcionalidades
- Registro de clientes
- Venta de herramientas
- Alquiler de herramientas
- Aplicación de descuentos
- Control de inventario

## 🧠 Lógica implementada
Se utiliza herencia mediante una clase abstracta `Herramienta` y una subclase `HerramientaElectrica`. Además, se implementan interfaces para manejar descuentos en compras y alquileres.

## 🛠️ Tecnologías
- Java
- Programación Orientada a Objetos
- Herencia e Interfaces

## 📂 Estructura
- Clase abstracta `Herramienta`
- Subclases de herramientas
- Clases `Compra` y `Alquiler`
